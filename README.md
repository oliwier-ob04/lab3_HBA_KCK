Aby uruchomić projekt lokalnie na linux, wykonaj poniższe kroki:

1. **Stwórz środowisko wirtualne:**
   ```bash
   python -m venv venv
   ```
2. **Aktywuj środowisko**
    ```bash
    source venv/bin/activate
    ```
3. **Zainstaluj wymagane biblioteki**
    ```bash
    pip install -r requirements.txt
    ```
